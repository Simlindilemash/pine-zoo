# Pine Zoo Website

A responsive multi-page website developed to provide visitors with information about **Pine Zoo**. The website allows users to explore different animals, discover attractions within the zoo, check the weather, and submit feedback through an easy-to-navigate interface.

This project was built to strengthen my front-end web development skills using HTML and CSS while practicing website navigation, page layout, and user-friendly design.

---

## Features

* Home page with zoo map
* Animal information pages
* Places and attractions pages
* Weather information page
* Feedback page
* Simple and intuitive navigation
* Clean and responsive layout

---

##  Technologies Used

* HTML5
* CSS3
